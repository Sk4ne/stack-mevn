# Operadores de consulta lógica

# $and (y)

Realiza la operación lógica "Y" en una array de expresiones y selecciona los documentos que cumplen todas las condiciones del array.

Syntax

{ $and: [ { <expression1> }, { <expression2> } , ... , { <expressionN> } ] }

Example

Mostrar a todas las personas cuya edad es mayor o igual a 18 pero distinta de 33

db.persona.find(
  {
    $and:[
      {
        age:{$ne:33}
      },
      {
        age:{$gte:18}
      }
    ]
  }
)

Example II

Mostrar a todas las personas cuya color de ojos es green y su edad es mayor o igual a 33

db.persona.find({
  $and:[
    {
      eyeColor: 'green'
    },
    {
      age: {$gte:33}
    }
  ]
})

Nota

El valor para el operador $and es un arreglo con cada una de las condiciones que debe cumplir.

# $or (o)

Este operador realiza la operación lógica "o" en un array de expresiones y selecciona los documentos que cumplen con al menos una condición.

Syntax

{ $or: [ { <expression1> }, { <expression2> }, ... ,{ <expressionN> } ] }

db.persona.find(
  {
    $or:[
      {
        eyeColor:'blue'
      },
      {
        eyeColor:'green'
      }
    ]
  }
)

# $not (no)

Inverts the effect of a query expression and returns documents that do not match the query expression.

Syntax

{ field: { $not: { <operator-expression> } } }

db.persona.find({age:{$not:{$gt:27}}});

# $nor

Une las cláusulas de la consulta con un NOR lógico devuelve todos los documentos que no coinciden con ambas cláusulas.

Syntax

{$nor:[{<expression1>},{<expression2>},...{ <expressionN> } ] }

db.persona.find({$nor:[{isActive:true},{eyeColor:'blue'}]});
